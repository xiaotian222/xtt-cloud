# 登录背景图片设置说明

## 📸 背景图片设置步骤

### 1. 保存背景图片
将你提供的山水风景图片保存为 `login-bg.jpg`，并放置在以下目录：
```
xtt-cloud-OA/front/public/login-bg.jpg
```

### 2. 图片要求
- **格式**: JPG 或 PNG
- **尺寸**: 建议 1920x1080 或更高分辨率
- **文件大小**: 建议小于 2MB，以保证加载速度

### 3. 效果预览
设置完成后，登录页面将具有以下效果：
- ✅ 全屏山水风景背景
- ✅ 半透明遮罩层，增强可读性
- ✅ 毛玻璃效果的登录框
- ✅ 优雅的阴影和边框效果
- ✅ 响应式设计，适配各种屏幕

### 4. 自定义选项
如果需要调整背景效果，可以修改 `src/assets/styles/login-bg.css` 文件：

```css
/* 调整遮罩层透明度 */
.login-container::before {
  background: rgba(0, 0, 0, 0.3); /* 0.3 是透明度，可以调整 */
}

/* 调整登录框透明度 */
.login-box {
  background: rgba(255, 255, 255, 0.95); /* 0.95 是透明度，可以调整 */
}
```

### 5. 备用方案
如果图片加载失败，系统会自动回退到渐变背景。

## 🎨 设计特色
- **自然风光**: 雪山、湖泊、蓝天白云的壮美景色
- **现代UI**: 毛玻璃效果和现代设计语言
- **用户体验**: 清晰的视觉层次和良好的可读性
- **性能优化**: 图片压缩和懒加载支持
